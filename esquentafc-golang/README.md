### Colinha de go
- go test ./... ==> executa os testes unitários no diretorio corrente
- go run => roda a aplicação (precisa ter um main.go)
`go run cmd/main.go`
- go build -o saida arquivo.go => compila a aplicação (precisa ter um main.go)
